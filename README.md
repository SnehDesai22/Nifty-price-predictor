# 📈 Nifty Price Prediction using Temporal Fusion Transformer (TFT)

This repository contains a Google Colab notebook that implements a Nifty 50 index price prediction model using the **Temporal Fusion Transformer (TFT)**. The project leverages deep learning-based time series forecasting techniques to make short-term predictions on stock market index data.

---

## 🚀 Features

- Temporal Fusion Transformer (TFT) model using PyTorch Forecasting
- Custom dataset preprocessing for financial time series
- Training, validation, and performance monitoring
- Scalable architecture for multivariate time series
- GPU acceleration using Colab

---

## 📁 Files

- `Nifty_price_prediction_tft.ipynb` - Main notebook with all code, from data loading to model training and evaluation.
